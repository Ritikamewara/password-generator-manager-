# 🔐 Password Manager & Generator

A Streamlit-based Password Manager and Password Generator built using Python.  
This application allows users to generate secure passwords, save custom credentials, view saved passwords, and delete stored passwords through an interactive web interface.

---

# 🚀 Features

- 🔐 Generate secure passwords
- ⚙️ Custom password options
- 💾 Save passwords permanently
- 👤 Store usernames/emails
- 📂 View saved passwords
- 🗑️ Delete passwords
- 🌐 Interactive Streamlit UI
- 📁 Local JSON storage

---

# 🛠️ Technologies Used

- Python
- Streamlit
- JSON
- Random
- String
- OS Module
